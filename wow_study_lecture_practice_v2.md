# WOW Study — Lecture & Practice: Full Implementation Instructions
> Based on screenshots: Image 1 = mode selection, Image 2 = lecture content with streaming

---

## The Problem Being Solved

Right now Lecture and Practice both render the same thing (text + voice).
After this implementation:

- **Lecture** = rich formatted reading experience with optional voice narration
- **Practice** = conversational task session where Claude asks questions one by one and waits for the student's answer

They are completely separate experiences with separate backends.

---

## PART 1 — LECTURE MODE

### What It Is

A beautifully formatted long-form article generated by Claude, streamed word-by-word as the student reads. Matches Image 2 exactly: numbered sections, tables, code blocks, bullet lists, bold terms, inline code. Optional voice button narrates the whole lecture.

---

### Backend: `POST /api/lesson/lecture`

This endpoint streams the lecture content so it appears word-by-word like in Image 2 (the "✦ Writing lecture..." indicator at the bottom).

```typescript
// routes/lesson-lecture.ts

router.post('/api/lesson/lecture', async (req, res) => {
  const { nodeId, studentId } = req.body;

  const student = await db.query('SELECT * FROM students WHERE id = $1', [studentId]);
  const memory = student.rows[0].memory;
  const node = memory.studyPath.find((n: RoadmapNode) => n.id === nodeId);

  if (!node) return res.status(404).json({ error: 'Node not found' });

  const studentName  = student.rows[0].name;
  const weakTopics   = memory.subjects?.[node.subject]?.weak ?? [];
  const level        = memory.subjects?.[node.subject]?.level ?? 'general';
  const learningStyle = memory.learningStyle ?? 'examples';

  // Set headers for streaming
  res.setHeader('Content-Type', 'text/plain; charset=utf-8');
  res.setHeader('Transfer-Encoding', 'chunked');
  res.setHeader('X-Content-Type-Options', 'nosniff');

  const stream = await anthropic.messages.create({
    model: 'claude-sonnet-4-6',
    max_tokens: 4000,
    stream: true,
    messages: [{
      role: 'user',
      content: buildLecturePrompt(node, studentName, level, weakTopics, learningStyle)
    }]
  });

  for await (const chunk of stream) {
    if (
      chunk.type === 'content_block_delta' &&
      chunk.delta.type === 'text_delta'
    ) {
      res.write(chunk.delta.text);
    }
  }

  res.end();
});

// ---------------------------------------------------------------
// LECTURE PROMPT — This shapes the entire lecture content
// ---------------------------------------------------------------
function buildLecturePrompt(
  node: RoadmapNode,
  studentName: string,
  level: string,
  weakTopics: string[],
  learningStyle: string
): string {
  return `
Write a complete, deeply informative lecture for a student.

Student: ${studentName}
Subject: ${node.subject}
Topic: ${node.topic}
Lesson title: ${node.title}
Level: ${level}
Learning style preference: ${learningStyle}
Known weak areas: ${weakTopics.join(', ') || 'none identified'}

CONTENT STRUCTURE — write exactly in this order:

1. One-line personal opener addressing the student by first name.
   Example: "Welcome back, Andi. Clean slate on this topic — let's build it from the ground up."

2. Section "1. Why this matters"
   Explain real-world consequences of understanding or misunderstanding this topic.
   3-5 sentences, conversational but substantive.

3. Section "2. Core idea"
   The fundamental concept explained from first principles.
   Include a TABLE if the content has multiple comparable items (regions, properties, types, rules).
   After the table, write 3-5 bullet points labeled "Formal rules to internalize:" with bold key terms inline.

4. Section "3. Worked example"
   A complete step-by-step worked example.
   - If the subject involves code (CS, programming): include a full code block in the correct language.
     Label each significant line with a comment // (A), // (B), etc.
     Then walk through each labeled line in a bullet list below the code block.
   - If the subject is math: show full working with each step on a new line.
   - If the subject is history/humanities: walk through a primary source or case study step by step.

5. Section "4. Common mistakes"
   2-3 bullet points. Each starts with the WRONG assumption in bold, then corrects it.
   Example: "**Stack memory persists after a function returns** — it does not..."

6. Section "5. Quick recap"
   3-5 bullet points. Crisp, memorable, exam-ready.

FORMATTING RULES — follow exactly:
- Use markdown headings: ## for section numbers, ### for subsections
- Use **bold** for key terms on first use
- Use \`inline code\` for any technical terms, function names, keywords
- Use triple backtick code blocks with language identifier for code
- Use markdown tables with | headers | like | this |
- Use bullet lists with - for informal lists, numbered lists 1. 2. 3. for sequential steps
- Use > blockquote for important warnings or key rules to remember
- NO generic filler sentences like "In this lesson we will..."
- NO outro like "I hope this was helpful"
- Write as a knowledgeable expert talking directly to the student

ALL content MUST be specifically about "${node.subject} — ${node.topic}".
Do NOT include content from other subjects or generic placeholder text.
`;
}
```

### Frontend: `/lesson/[nodeId]/lecture/page.tsx`

```tsx
'use client';
import { useEffect, useRef, useState } from 'react';
import { useParams, useRouter } from 'next/navigation';
import ReactMarkdown from 'react-markdown';    // npm install react-markdown
import remarkGfm from 'remark-gfm';            // npm install remark-gfm (tables support)

const API = 'https://hackatonhoiv-production.up.railway.app';

export default function LecturePage() {
  const { nodeId } = useParams<{ nodeId: string }>();
  const router     = useRouter();

  const [content, setContent]       = useState('');     // streamed markdown
  const [isStreaming, setIsStreaming] = useState(true);
  const [node, setNode]             = useState<RoadmapNode | null>(null);
  const [voiceMode, setVoiceMode]   = useState(false);
  const [voiceScript, setVoiceScript] = useState('');
  const scrollRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    const studentId = localStorage.getItem('studentId')!;

    // 1. Fetch node metadata first (for top bar)
    fetch(`${API}/api/lesson/node/${nodeId}?studentId=${studentId}`)
      .then(r => r.json())
      .then(d => setNode(d.node));

    // 2. Stream lecture content
    streamLecture(studentId);
  }, [nodeId]);

  const streamLecture = async (studentId: string) => {
    const res = await fetch(`${API}/api/lesson/lecture`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ nodeId, studentId }),
    });

    const reader  = res.body!.getReader();
    const decoder = new TextDecoder();
    let accumulated = '';

    while (true) {
      const { done, value } = await reader.read();
      if (done) { setIsStreaming(false); break; }

      const chunk = decoder.decode(value, { stream: true });
      accumulated += chunk;
      setContent(accumulated);

      // Auto-scroll during streaming
      scrollRef.current?.scrollTo({ top: scrollRef.current.scrollHeight, behavior: 'smooth' });
    }

    // Store plain text for voice (strip markdown)
    setVoiceScript(accumulated.replace(/#{1,6}\s/g, '').replace(/[*`_|]/g, ''));
  };

  return (
    <div className="lecture-page">
      {/* Top bar — matches Image 2 */}
      <LessonTopBar
        node={node}
        activePhase="lecture"
        onBack={() => router.push(`/lesson/${nodeId}`)}
        voiceMode={voiceMode}
        onVoiceToggle={() => setVoiceMode(v => !v)}
      />

      {/* Content area */}
      <div className="lecture-scroll-area" ref={scrollRef}>
        <div className="lecture-content">

          {/* Section label + title — matches Image 2 top */}
          {node && (
            <>
              <p className="lecture-label">LECTURE</p>
              <h1 className="lecture-title">{node.title}</h1>
            </>
          )}

          {/* Streamed markdown */}
          <ReactMarkdown
            remarkPlugins={[remarkGfm]}
            components={MARKDOWN_COMPONENTS}  // see below
          >
            {content}
          </ReactMarkdown>

          {/* Streaming indicator — bottom left, matches Image 2 */}
          {isStreaming && (
            <div className="streaming-indicator">
              <span className="streaming-dot">✦</span>
              <span>Writing lecture...</span>
            </div>
          )}
        </div>
      </div>

      {/* Voice player (shown when voiceMode is on) */}
      {voiceMode && (
        <VoicePlayer
          nodeId={nodeId}
          script={voiceScript}
          isReady={!isStreaming}
        />
      )}

      {/* Bottom bar — question input + Practice CTA */}
      {!voiceMode && (
        <LectureBottomBar
          nodeId={nodeId}
          onGoToPractice={() => router.push(`/lesson/${nodeId}/practice`)}
        />
      )}

      {/* Stop streaming button (bottom-right circle, matches Image 2) */}
      {isStreaming && (
        <button
          className="stop-streaming-btn"
          onClick={() => setIsStreaming(false)}
          title="Stop generating"
        >
          ■
        </button>
      )}
    </div>
  );
}
```

```tsx
// components/lesson/lecture/LectureBottomBar.tsx

export function LectureBottomBar({ nodeId, onGoToPractice }) {
  const [question, setQuestion] = useState('');
  const [answer, setAnswer]     = useState('');
  const [loading, setLoading]   = useState(false);

  const askQuestion = async () => {
    if (!question.trim()) return;

    const res = await fetch(`${API}/api/tutor/message`, {
      method: 'POST',

          <p>{answer}</p>
          <button onClick={() => setAnswer('')}>✕</button>

        <input
          className="question-input"
          placeholder="Ask a question about the lecture..."
          value={question}
          onChange={e => setQuestion(e.target.value)}

        {/* Practice CTA */}

        </button>
      </div>

  );
```

## PART 2 — PRACTICE MODE

### What It Is

A **conversational task session**. Claude asks one question at a time and waits for the student's answer. It is NOT a quiz with pre-loaded options — Claude evaluates free-text answers, gives feedback, offers hints, and moves to the next question based on whether the student got it right.

This feels like a tutor sitting across from the student asking questions, not a multiple-choice test.

---

### Backend: `POST /api/lesson/practice/start`

Creates a practice session and returns the first question.

```typescript
router.post('/api/lesson/practice/start', async (req, res) => {
  const { nodeId, studentId } = req.body;

  const student = await db.query('SELECT * FROM students WHERE id = $1', [studentId]);
  const memory  = student.rows[0].memory;
  const node    = memory.studyPath.find((n: RoadmapNode) => n.id === nodeId);

  // Generate the full practice session plan upfront
  // Store it in memory so subsequent /ask calls can reference it
  const sessionPlan = await generatePracticeSession(node, memory);

  // Save session to DB
  const sessionId = crypto.randomUUID();
  await db.query(
    `INSERT INTO practice_sessions (id, student_id, node_id, plan, current_task, score, created_at)
     VALUES ($1, $2, $3, $4, 0, 0, NOW())`,
    [sessionId, studentId, nodeId, JSON.stringify(sessionPlan)]
  );

  res.json({
    sessionId,
    totalTasks: sessionPlan.tasks.length,
    firstQuestion: {
      taskIndex:   0,
      type:        sessionPlan.tasks[0].type,
      question:    sessionPlan.tasks[0].question,
      context:     sessionPlan.tasks[0].context ?? null,
      hint:        null,   // not shown until requested
      xpReward:    sessionPlan.tasks[0].xpReward,
    }
  });
});

async function generatePracticeSession(node: RoadmapNode, memory: any) {
  const response = await anthropic.messages.create({
    model: 'claude-sonnet-4-6',
    max_tokens: 2500,
    messages: [{
      role: 'user',
      content: `Generate a practice session for a student.

Subject: ${node.subject}
Topic: ${node.topic}
Lesson: ${node.title}
Level: ${memory.subjects?.[node.subject]?.level ?? 'general'}
Weak areas: ${(memory.subjects?.[node.subject]?.weak ?? []).join(', ') || 'none'}

Return ONLY valid JSON with this structure:
{
  "tasks": [
    {
      "type": "conceptual" | "application" | "debugging" | "explanation",
      "question": "The question to ask the student",
      "context": "Optional: code snippet, scenario, table, or formula for reference",
      "correctAnswer": "Full model answer",
      "keyPoints": ["point 1", "point 2", "point 3"],
      "hint": "One-sentence hint that doesn't give away the answer",
      "followUp": "A follow-up question to ask if student gets it right",
      "xpReward": 30
    }
  ]
}

Rules:
- exactly 4 tasks
- task 1: conceptual — tests if student understands the core idea
- task 2: application — give them a scenario or code to analyze
- task 3: debugging/error — show something WRONG and ask them to identify/fix it
- task 4: explanation — ask them to explain the topic in their own words
- ALL tasks MUST be about "${node.subject} — ${node.topic}" only
- correctAnswer must be complete enough to grade a student's free-text response against
- context: for code subjects include actual code; for math include formula; for history include a quote or date`
    }]
  });

  const text = response.content[0].type === 'text' ? response.content[0].text : '';
  return JSON.parse(text.replace(/```json|```/g, '').trim());
}
```

### Backend: `POST /api/lesson/practice/answer`

Student submits their answer. Claude grades it and returns feedback + next question.

```typescript
router.post('/api/lesson/practice/answer', async (req, res) => {
  const { sessionId, studentId, answer } = req.body;

  // Load session
  const session = await db.query(
    'SELECT * FROM practice_sessions WHERE id = $1 AND student_id = $2',
    [sessionId, studentId]
  );
  if (!session.rows[0]) return res.status(404).json({ error: 'Session not found' });

  const plan        = session.rows[0].plan;
  const taskIndex   = session.rows[0].current_task;
  const currentTask = plan.tasks[taskIndex];
  const isLastTask  = taskIndex === plan.tasks.length - 1;

  // Grade the answer with Claude Haiku (fast)
  const gradeRes = await anthropic.messages.create({
    model: 'claude-haiku-4-5-20251001',
    max_tokens: 400,
    messages: [{
      role: 'user',
      content: `Grade this student answer.

Question: ${currentTask.question}
Model answer key points: ${currentTask.keyPoints.join(' | ')}
Full correct answer: ${currentTask.correctAnswer}
Student's answer: ${answer}

Return ONLY valid JSON:
{
  "score": 0 | 0.5 | 1,
  "isCorrect": boolean,
  "feedback": "2-3 sentence feedback. Be encouraging but honest. Point out what was right and what was missing.",
  "missedConcept": "null or the key concept they missed",
  "xpAwarded": number (0 if score 0, 15 if 0.5, 30 if 1)
}`
    }]
  });

  const gradeText = gradeRes.content[0].type === 'text' ? gradeRes.content[0].text : '';
  const grade = JSON.parse(gradeText.replace(/```json|```/g, '').trim());

  // Update session in DB
  const newScore = session.rows[0].score + grade.xpAwarded;
  const nextTask = isLastTask ? null : taskIndex + 1;

  await db.query(
    `UPDATE practice_sessions
     SET current_task = $1, score = $2, completed = $3
     WHERE id = $4`,
    [nextTask ?? taskIndex, newScore, isLastTask, sessionId]
  );

  // Award XP to student
  if (grade.xpAwarded > 0) {
    await db.query(
      'UPDATE students SET xp = xp + $1 WHERE id = $2',
      [grade.xpAwarded, studentId]
    );
  }

  res.json({
    feedback:   grade.feedback,
    isCorrect:  grade.isCorrect,
    score:      grade.score,
    xpAwarded:  grade.xpAwarded,
    correctAnswer: grade.score < 1 ? currentTask.correctAnswer : null, // only reveal if wrong
    followUp:   grade.isCorrect ? currentTask.followUp : null,         // bonus question if correct

    isLastTask,
    nextQuestion: isLastTask ? null : {
      taskIndex:  nextTask,
      type:       plan.tasks[nextTask!].type,
      question:   plan.tasks[nextTask!].question,
      context:    plan.tasks[nextTask!].context ?? null,
      xpReward:   plan.tasks[nextTask!].xpReward,
    },
    totalXpEarned: newScore,
  });
});
```

```tsx
'use client';
import { useEffect, useState } from 'react';
import { useParams, useRouter } from 'next/navigation';

const API = 'https://hackatonhoiv-production.up.railway.app';

  // After submission
  const [feedback, setFeedback]         = useState('');
  const [isCorrect, setIsCorrect]       = useState(false);
  const [xpAwarded, setXpAwarded]       = useState(0);
  const [correctAnswer, setCorrectAnswer] = useState<string | null>(null);
  const [followUp, setFollowUp]         = useState<string | null>(null);

  // Hint
  const [hint, setHint]           = useState<string | null>(null);
  const [hintLoading, setHintLoading] = useState(false);

  // Session totals
  const [totalXp, setTotalXp]     = useState(0);
  const [correctCount, setCorrectCount] = useState(0);

  useEffect(() => {
    const studentId = localStorage.getItem('studentId')!;
    startSession(studentId);
  }, [nodeId]);

  const startSession = async (studentId: string) => {
    const res = await fetch(`${API}/api/lesson/practice/start`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ nodeId, studentId }),
    });
    const data = await res.json();

    setSessionId(data.sessionId);
    setTotalTasks(data.totalTasks);
    setCurrentQuestion(data.firstQuestion);
    setState('question');
  };

  const submitAnswer = async () => {
    if (!answer.trim() || loading) return;
    setLoading(true);

    const res = await fetch(`${API}/api/lesson/practice/answer`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        sessionId,
        studentId: localStorage.getItem('studentId'),
        answer,
      }),
    });
    const data = await res.json();

    setFeedback(data.feedback);
    setIsCorrect(data.isCorrect);
    setXpAwarded(data.xpAwarded);
    setCorrectAnswer(data.correctAnswer);
    setFollowUp(data.followUp);
    setTotalXp(data.totalXpEarned);
    if (data.isCorrect) setCorrectCount(c => c + 1);

    setState(data.isLastTask ? 'complete' : 'submitted');
    setLoading(false);
    setHint(null);
  };

  const requestHint = async () => {
    setHintLoading(true);
    const res = await fetch(`${API}/api/lesson/practice/hint`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ sessionId, studentId: localStorage.getItem('studentId') }),
    });
    const data = await res.json();
    setHint(data.hint);
    setHintLoading(false);
  };

  const goToNext = () => {
    setAnswer('');
    setFeedback('');
    setCorrectAnswer(null);
    setFollowUp(null);
    setHint(null);
    // next question data came in the answer response
    setTaskIndex(t => t + 1);
    setState('question');
  };

  return (
    <div className="practice-page">
      <LessonTopBar
        node={node}
        activePhase="practice"
        onBack={() => router.push(`/lesson/${nodeId}`)}
        showVoice={false}
      />

      <div className="practice-content">

        {/* Task progress dots */}
        <TaskProgressDots
          total={totalTasks}
          current={taskIndex}
          state={state}
        />

        {/* STATES */}
        {state === 'loading' && <PracticeLoading />}

        {(state === 'question' || state === 'submitted') && currentQuestion && (
          <PracticeTaskCard
            question={currentQuestion}
            taskIndex={taskIndex}
            totalTasks={totalTasks}
            answer={answer}
            onAnswerChange={setAnswer}
            onSubmit={submitAnswer}
            onHint={requestHint}
            hint={hint}
            hintLoading={hintLoading}
            loading={loading}
            submitted={state === 'submitted'}
            feedback={feedback}
            isCorrect={isCorrect}
            xpAwarded={xpAwarded}
            correctAnswer={correctAnswer}
            followUp={followUp}
            onNext={goToNext}
          />
        )}

        {state === 'complete' && (
          <PracticeResults
            correctCount={correctCount}
            totalTasks={totalTasks}
            totalXp={totalXp}
            nodeId={nodeId}
            node={node}
            onReturn={() => router.push(`/dashboard/${encodeURIComponent(node?.subject ?? '')}`)}
          />
        )}

      </div>
    </div>
  );
}
```
